Tentang Saya

Yuk, belajar web programing bersama aku

Saya adalah seorang mahasiswa Informatika di Universitas AMIKOM Yogyakarta dengan keahlian dalam pembuatan dan pengembangan website. Keahlian ini saya peroleh melalui pengalaman belajar, pengerjaan proyek Tugas Akhir (TA), magang di instansi swasta, serta partisipasi dalam program pertukaran pelajar. Saya memiliki pengalaman bekerja sama dengan mitra untuk menyelesaikan proyek pembuatan website. Saya adalah pribadi yang jujur, disiplin dalam mengelola waktu, dan siap berkontribusi sebagai bagian dari tim yang dinamis dalam lingkungan kerja profesional.

Portfolio

Project yg pernah saya kerjakan

Sebagai seorang pengembang web, saya telah mengerjakan beberapa proyek yang membantu saya memahami berbagai aspek pengembangan website. Saya fokus pada fungsionalitas, pengalaman pengguna yang optimal, dan desain yang responsif. Berikut adalah beberapa proyek yang telah saya kerjakan:

SiMaPa

Website ini adalah sistem informasi manajemen presensi sekolah yang digunakan oleh para guru untuk mempresensi muridnya.

CosrentD

Website ini adalah rental kostum karakter yang digunakan oleh para mitra dan client untuk melakukan transaksi sewa kostum.

Bibit Mall

Website ini adalah sistem informasi pendataan bibit tanaman durian yang digunakan oleh mitra usaha untuk mendisplaykan produknya.

LearnEasy

Website ini merupakan website penyedia kursus pembelajaran yang digunakan oleh para pencari kursus untuk memudahkan pengunjung untuk mencari kursus terpercaya
